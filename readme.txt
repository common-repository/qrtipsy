A simple Plugin to Convert your links in Qr Codes and Show them in six classic coloured tooltips.

== Description ==

For All those Users Who want their visitors to visit their site via mobile phone with just a scan, then this is a 
**PERFECT** plugin to use.

*Features*

1. Uses Google Chart Apis to create QrCodes.
2. Uses [colorTip jQuery plugin](http://tutorialzine.com/2010/07/colortips-jquery-tooltip-plugin/).
3. Also use [js-uri](https://code.google.com/p/js-uri/) to get the relative links right.
4. Originally Implemented by [Konstantin Kovshenin](http://kovshenin.com/) and converted to plugin by [Agnel Waghela](http://goo.gl/yvxAb).

**Dont forget to Take look at the Screenshots**

== Installation ==

1. Upload Plugin `qrtipsy` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go To your QrTipsy Dashboard and Customise the settings as per your needs. If you have any problem then see the documentation. If you have any queries then go to my [blog](http://agnelwaghela.wordpress.com/) and Leave a Comment there and We'll get back to you as soon as possible.
4. Then To use qrtipsy in your blog go to the post/ page editor use the shortcode inserting tool, which is tinymce button on the Visual Editor paste the link and Hit insert shortcode button and there you go.
